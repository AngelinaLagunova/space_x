# Space_X

### Как запустить проект
1. Установить Docker и Git
2. `git clone https://github.com/AngelinaLagunova/space_x.git`
3. `docker compose up -d`

Админ-панель доступна по адресу http://<ваш IP>:8000/admin/

Frontend-часть разрабатывалась на React.js  
Backend-часть разрабатывалась на Django
